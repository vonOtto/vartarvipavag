# Content Style Guide — På Spåret Party Edition

## Syfte

Detta dokument definierar språkregler och tonalitet för allt AI-genererat innehåll i På Spåret Party Edition. Målet är att skapa innehåll som låter naturligt svenskt, är roligt utan att bli flummigt, och engagerar spelarna.

---

## Svenska språkregler

### Undvik anglicismer

Skriv svensk svenska, inte direktöversatt engelska.

| ❌ Undvik | ✅ Använd istället |
|-----------|-------------------|
| realisera | inse, förstå, genomföra |
| facilitera | underlätta, möjliggöra |
| implementera | införa, genomföra |
| approximera | uppskatta, närma sig |
| relatera till | ha samband med, handla om |
| fokusera på | koncentrera sig på |

### Aktiva verb framför passiva konstruktioner

Använd aktiva formuleringar som ger liv åt texten.

| ❌ Passivt | ✅ Aktivt |
|-----------|----------|
| Staden är känd för att ha många kanaler | Staden har många kanaler |
| Detta land är berömt för sina fjordar | Detta land har fantastiska fjordar |
| Här finns ett museum som är dedicerat till konst | Här finns ett konstmuseum |
| Byggnaden är lokaliserad vid floden | Byggnaden ligger vid floden |

### Konkreta exempel framför vaga beskrivningar

Ge konkreta exempel som är lättare att förstå och komma ihåg.

| ❌ Vagt | ✅ Konkret |
|---------|-----------|
| Ett välkänt möbelföretag | IKEA |
| En känd musikstreamingapp | Spotify |
| Ett stort mode-företag | H&M |
| Många internationella företag | Över 50 globala huvudkontor |

### Naturligt språk

Skriv som en svensk skulle prata naturligt, inte som en lärobok.

| ❌ Onaturligt | ✅ Naturligt |
|--------------|--------------|
| Detta fantastiska land är känt för sina många sevärdheter | Detta land har många sevärdheter |
| Här kan man uppleva den vackra naturen | Här finns vacker natur |
| Destinationen erbjuder många möjligheter | Här kan du göra mycket |
| Den här platsen hyser många kulturella attraktioner | Här finns mycket kultur |

### Inkluderande pronomen

Använd "du/ni/här" framför opersonligt "man".

| ❌ Opersonligt | ✅ Inkluderande |
|---------------|-----------------|
| Här kan man besöka museet | Här kan du besöka museet |
| Man hittar många restauranger | Du hittar många restauranger |
| I denna stad kan man uppleva | I denna stad upplever du |

## Tonalitet

### Grundprinciper

**Rolig, smart, inte för tung**

- Lek med språket, men var inte fånig
- Undvik fluff ("fantastiska", "vackra", "underbara") om det inte tillför
- Var entusiastisk men inte överdriven
- Smart utan att vara nedlåtande

### Ledtrådar (clues)

**Stil:** Börja ofta med "Här..." för att skapa känsla av att man är på plats.

**Progressiv svårighetsgrad:**
- Nivå 10: Subtila, indirekta fakta (inga uppenbara markörer)
- Nivå 8: Fortfarande svårt, kräver god allmänbildning
- Nivå 6: Medelnivå, mer igenkännbart
- Nivå 4: Lättare, kända fakta
- Nivå 2: Uppenbart, kan nämna stadens namn

**Exempel (Stockholm):**

```
Nivå 10: "Här finns 14 öar sammanbundna av över 50 broar, och staden byggdes där Mälaren möter Östersjön."
Nivå 8: "Nobelpriset delas ut i stadshuset här varje december."
Nivå 6: "Här ligger Gamla Stan, en av Europas bäst bevarade medeltida stadskärnor."
Nivå 4: "Vasamuseet här hyser ett 1600-talsskepp som sjönk på jungfruresa 1628."
Nivå 2: "Sveriges huvudstad och största stad, känd för sina vackra byggnader och vattenvägar."
```

**Före/efter exempel:**

| ❌ Före | ✅ Efter |
|---------|----------|
| Detta fantastiska land är känt för sina många vackra kanaler | Detta land har över 100 kilometer kanaler |
| Här finns ett museum som är dedicerat till konst | Här ligger världens största konstmuseum |
| Staden är känd för att ha faciliterat många internationella möten | Staden har varit värd för FN-toppmöten sedan 1950 |

### Följdfrågor (followups)

**Stil:** Tydliga frågor utan att nämna destinationens namn.

**Formuleringar:**
- "Vilken flod flödar genom denna stad?"
- "Vilket år invigdes [byggnad] här?"
- "Vad heter sjön som möter [hav] vid denna stad?"
- "Hur många öar består denna stad av?"

**Exempel:**

```
✅ Bra: "Vilket år sjönk skeppet Vasa som nu visas i museet här?"
❌ Dåligt: "Vilket år sjönk skeppet Vasa som nu visas i Vasamuseet i Stockholm?"

✅ Bra: "Vad heter sjön som möter Östersjön vid denna stad?"
❌ Dåligt: "Vad heter sjön som möter Östersjön vid Stockholm?"
```

### Banter (feedback till spelare)

**Rätt svar:** Variera mellan entusiastisk och bekräftande.

```
✅ Bra variation:
- "Helt rätt!"
- "Precis!"
- "Det stämmer!"
- "Skarpt!"
- "Just det!"
- "Toppen!"
- "Bra jobbat!"

❌ Undvik:
- "Excellent!" (engelska)
- "Spot on!" (engelska)
- "Mycket väl svarat!" (för formellt)
```

**Fel svar:** Uppmuntrande utan att vara nedlåtande.

```
✅ Bra variation:
- "Aj då, inte rätt!"
- "Fortsätt gissa!"
- "Det var nära!"
- "Nästan!"
- "Tyvärr fel den här gången"

❌ Undvik:
- "Tyvärr var det inte korrekt" (för formellt)
- "Det var ingen lyckträff" (konstigt)
- "Försök igen nästa gång" (för tråkigt)
```

### Instruktioner

**Stil:** Tydliga, direkta, energiska.

```
✅ Bra:
- "Tryck på bromsen när du vet svaret!"
- "Dags för följdfråga!"
- "Svaret är låst!"

❌ Undvik:
- "Vänligen tryck på bromsen när du känner dig redo att svara" (för formellt)
- "Nu kommer vi att presentera en följdfråga" (för stelt)
```

## Vanliga misstag att undvika

### 1. För mycket fluff

| ❌ Undvik | ✅ Bättre |
|-----------|----------|
| Denna fantastiska stad med sina vackra byggnader | Denna stad |
| Det här underbara landet är känt för | Detta land har |
| Här kan du uppleva den magnifika naturen | Här finns vacker natur |

### 2. Onödiga ord

| ❌ Undvik | ✅ Bättre |
|-----------|----------|
| Här finns det många | Här finns många |
| Detta är ett land som | Detta land |
| Man kan säga att | (ta bort helt) |

### 3. Formella konstruktioner

| ❌ Undvik | ✅ Bättre |
|-----------|----------|
| Det förekommer många | Det finns många |
| Platsen hyser | Här finns |
| Staden erbjuder | Du hittar |

### 4. Engelska uttryck

| ❌ Undvik | ✅ Bättre |
|-----------|----------|
| Spot on! | Precis! |
| Excellent! | Utmärkt! |
| Amazing! | Fantastiskt! |

## Checklista för content review

När du granskar AI-genererat innehåll, kontrollera:

- [ ] Inga anglicismer (realisera, facilitera, etc.)
- [ ] Aktiva verb (inte "är känd för att ha")
- [ ] Konkreta exempel (inte vaga beskrivningar)
- [ ] Naturligt språk (inte "Detta fantastiska...")
- [ ] Inkluderande pronomen (du/ni/här, inte "man")
- [ ] Rätt tonalitet (rolig och smart, inte tung)
- [ ] Inga onödiga fluff-ord
- [ ] Svenska uttryck (inte "Spot on!")
- [ ] Progressiv svårighetsgrad för ledtrådar
- [ ] Anti-leak (inget läckage av destinationsnamn i tidiga ledtrådar/följdfrågor)

## Användning i AI-prompts

Lägg till följande i alla Claude API-calls för innehållsgenerering:

```typescript
const SWEDISH_LANGUAGE_RULES = `
VIKTIGT - Svenska språkregler:
- Använd aktiva verb (inte "är känd för att ha" → använd "har")
- Undvik onödigt fluff ("fantastiska", "vackra" om det inte tillför)
- Konkreta exempel > vaga beskrivningar ("IKEA" > "möbelföretag")
- Naturligt språk som en svensk skulle säga
- Inga anglicismer (realisera → inse, facilitera → underlätta)
- Inkluderande ton: "Här kan du..." (inte "Man kan...")
- Rolig och smart, inte för tung eller formell
`;
```

Inkludera sedan i system prompt:

```typescript
const systemPrompt = `${SWEDISH_LANGUAGE_RULES}\n\n${restOfSystemPrompt}`;
```
