Utför en säker git commit med automatiska säkerhetskontroller.

Steg:
1. Kör git status
2. Klassificera filer som SAFE/UNSAFE/LOCAL
3. Verifiera .gitignore
4. Add endast SAFE filer
5. Visa git diff --staged för granskning
6. Skapa commit med conventional commits format
7. Uppdatera docs/git-report.md
8. Returnera commit hash och sammanfattning
